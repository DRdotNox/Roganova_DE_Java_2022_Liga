Добро пожаловать в справку:

Основные команды для использования данного сервиса:
test — заполняет базу данных тестовыми значениями

add — добавляет запись в базу
	add user — добавит пустого пользователя без имени
	add user Гарри Джеймосвич Поттер — добавит пользователя с соответствующим именем
	
	Для создания задания используйте следующие ключи:

	-h для заголовока
	-desc для описания
	-userId для указания пользователя
	-d для даты
	
	Некоторые данные можно опускать,но сам порядок ОБЯЗАТЕЛЕН ДЛЯ СОБЛЮДЕНИЯ
	
	add task — создаст задание без описания даты и заголовка, id юзера будет равен -1, а дедлайн текущей дате
	add task -h Пастафарианство для чайников -userId 3 — описание и дедлайн опущены, но порядок сохранен
	add task -h Сделать больно 101 -desc Распарсить эту команду правильно -userId 3 -d 11.03.1997 —  полная версия команды

edit — редактирование
	За раз можно редактировать только одно поле!
	
	edit user 3 name Сергей Разумовский — поменяет имя пользователя с id = 6

	edit task 6 -desc Новое описание — поменяет описание у задания с id = 6
	
	для смены статуса задачи используется ключ -s
	edit task 6 -s IN_PROGRESS 
	
	Доступные статусы:
	NEW
	IN_PROGRESS
	DONE
	
get — получение записей
	get user 3 — вернет пользователя с id = 3 (и список его заданий!)
	get users — вернет список всех пользователей
	
	Для заданий аналогично:
	get task 3
	get tasks

delete — удаление записей
	delete user 3 — удалит пользователя с id = 3
	delete users all — удалит всех пользователей
	
	delete task 3 — удалит задание с id = 3
	delete tasks all — удалит все задания

save — сохраняет данные в новые csv файлы
